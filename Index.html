Conhecimentos

HTML

O HTML (HyperText Markup Language) é um conjunto estruturado de instruções, conhecidas por etiquetas ou tags, que dizem a um browser como publicar uma página web, ou seja, o browser interpreta essas etiquetas e desenha a página no ecrã.

CSS

CSS é utilizado para definir a apresentação (aparência) em páginas da internet que adotam para o seu desenvolvimento linguagens de marcação (como XML, HTML e XHTML). O CSS define como serão exibidos os elementos contidos no código de uma página da internet e sua maior vantagem é efetuar a separação entre o formato e o conteúdo de um documento.

Git/GitHub

O GitHub é um site que abriga um software de controle de versão de desenvolvimento através do sistema Git.
Através desse sistema, duas ou mais pessoas podem editar arquivos existentes ou criar novos sem arriscar que suas alterações sejam apagadas pelas modificações de outros membros da equipe. Isso é útil para os mais diferentes tipos de projetos

ABAP

ABAP é uma linguagem de programação de alto nível desenvolvida pela empresa de software SAP. É a principal linguagem utilizada no produto mais conhecido desta empresa, o SAP R/3, um software ERP.

MySQL

O MySQL é um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface. É atualmente um dos sistemas de gerenciamento de bancos de dados mais populares da Oracle Corporation.
